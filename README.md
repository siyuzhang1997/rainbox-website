## 🌟 项目简介

RainBox 数字科技公司官方网站，展示公司核心业务、AI Agent 产品、团队介绍和联系方式。

## 🎨 设计特色

- **品牌色彩**：科技蓝 (#2196F3) + 活力橙 (#FF9800)
- **现代化设计**：渐变背景、流畅动画、响应式布局
- **技术栈**：HTML5 + Tailwind CSS + Vanilla JavaScript
- **AI 生成**：由 [With](https://with.woa.com/) AI 平台生成

## 🛠️ 本地开发

```bash
# 克隆仓库
git clone https://github.com/siyuzhang1997/rainbox-website.git

# 进入目录
cd rainbox-website

# 使用任意 HTTP 服务器运行
python3 -m http.server 8000
# 或
npx serve

# 访问 http://localhost:8000
```
