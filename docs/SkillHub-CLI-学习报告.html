Frontend Design 技能核心优势

根据技能文档分析，该技能特别强调：

🎨 设计思维

  • 目的导向: 理解界面解决的问题和用户群体
  • 风格定位: 选择明确的美学方向（极简、复古未来、奢华、工业风等）
  • 差异化: 创造令人难忘的独特体验

🎯 美学指南

  1. 排版: 选择独特、有个性的字体，避免 Arial、Inter 等通用字体
  2. 配色: 使用 CSS 变量保持一致性，主色调配合鲜明的强调色
  3. 动画: 使用 CSS 动画和 Motion 库创建微交互
  4. 空间布局: 非对称、重叠、对角线流动、打破网格的元素
  5. 背景细节: 渐变网格、噪点纹理、几何图案、分层透明度

⚠️ 避免的通用美学

  • ❌ 过度使用的字体（Inter、Roboto、Arial）
  • ❌ 陈词滥调的配色（特别是白底紫色渐变）
  • ❌ 可预测的布局和组件模式
  • ❌ 缺乏上下文特色的千篇一律设计
