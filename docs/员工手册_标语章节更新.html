第一章：公司文化与价值观（更新版）

📌 更新说明

  • 更新日期: 2026年3月22日
  • 更新内容: 新增公司标语章节，更新文化价值观表述
  • 适用版本: 员工手册 v3.0 及以后版本

1.1 公司标语

🏢 官方标语

"科技养虾，用心养人"

这是我们的使命宣言，也是每位员工的行动指南。

📖 标语释义

科技养虾

我们致力于用前沿科技革新水产养殖产业：
- 运用 AI、IoT、大数据等技术提升养殖效率
- 通过智能化手段保障水产品质量与安全
- 推动传统产业向数字化、智能化转型
- 成为水产养殖科技领域的创新引领者

用心养人

我们像呵护养殖对象一样精心培养每一位员工：
- 职业成长：提供系统化培训与清晰的发展路径
- 工作环境：营造舒适、高效、创新的工作氛围
- 福利保障：提供具有竞争力的薪酬与全面的福利
- 人文关怀：关注员工身心健康与工作生活平衡

1.2 核心价值观

基于标语精神，公司倡导以下核心价值观：

🔬 创新驱动

  • 持续探索新技术在养殖领域的应用
  • 鼓励试错，包容失败，奖励创新
  • 跨部门协作，激发创意碰撞

🎯 专业卓越

  • 深耕水产养殖与科技融合领域
  • 追求产品与服务的极致品质
  • 持续学习，保持专业领先性

💚 以人为本

  • 尊重每一位员工的价值与贡献
  • 关注员工的全面发展与幸福感
  • 构建开放、包容、有温度的文化

🤝 协作共赢

  • 团队利益高于个人利益
  • 跨部门高效协同，打破信息孤岛
  • 与合作伙伴共建生态，实现多方共赢

🌱 可持续发展

  • 关注环境保护与资源可持续利用
  • 推动产业健康发展，承担社会责任
  • 为员工创造长期稳定的职业发展平台

1.3 标语在工作中的体现

对员工的期望

践行"科技养虾"：

  • ✅ 积极学习新技术，提升专业能力
  • ✅ 在工作中寻找创新优化的机会
  • ✅ 关注行业动态，保持技术敏感度
  • ✅ 将科技思维融入日常工作流程

践行"用心养人"：

  • ✅ 尊重同事，积极协作
  • ✅ 主动分享知识与经验
  • ✅ 关心团队成员的成长与困难
  • ✅ 营造积极向上的团队氛围

公司对员工的承诺

基于"用心养人"，公司承诺：

1. 职业发展支持
- 每年至少 40 小时的带薪培训时间
- 个性化职业发展规划（IDP）
- 内部晋升优先政策
- 导师制与技能传承计划

2. 工作环境保障
- 符合人体工学的办公设备
- 灵活的工作时间安排（弹性工作制）
- 远程办公支持（根据岗位特性）
- 定期团队建设活动

3. 薪酬福利体系
- 行业具有竞争力的薪酬水平
- 年度绩效奖金与项目奖金
- 五险一金 + 补充商业保险
- 年度体检、健康管理服务
- 带薪年假、病假、婚假、产假等

4. 人文关怀措施
- 员工生日礼物与祝福
- 重大节日福利与活动
- 员工困难帮扶基金
- 心理健康咨询服务
- 家庭关怀计划（子女教育、父母健康等）

1.4 文化落地机制

📊 文化评估体系

  • 每季度开展员工满意度调查
  • 将文化践行纳入绩效考核（占比 20%）
  • 设立"文化大使"表彰机制

🎤 文化传播渠道

  • 新员工入职培训必讲标语文化
  • 每月文化主题活动
  • 内部文化墙与宣传物料
  • 管理层以身作则，率先垂范

💬 反馈与改进

  • 开通文化建议邮箱：culture@company.com
  • 每季度召开文化建设座谈会
  • 根据反馈持续优化文化举措

1.5 员工行为准则

基于标语精神，所有员工应遵守以下行为准则：

✅ 应该做的事：

  • 积极拥抱技术创新，提升工作效率
  • 尊重同事，平等对待每一个人
  • 主动沟通，及时反馈问题
  • 关注团队目标，而非个人得失
  • 保持学习热情，持续自我提升
  • 维护公司声誉，传播正能量

❌ 不应该做的事：

  • 抵触新技术、新方法的应用
  • 歧视、骚扰或不尊重他人
  • 隐瞒问题，推卸责任
  • 传播负面情绪或不实信息
  • 泄露公司机密或客户信息
  • 损害公司利益或品牌形象

1.6 违规处理

对于严重违反标语精神与公司价值观的行为，公司将根据《员工奖惩制度》进行处理：

  • 轻微违规：口头警告 + 文化培训
  • 一般违规：书面警告 + 绩效扣分
  • 严重违规：降职降薪或解除劳动合同
