📋 管理框架

1. 战略管理

OODA 决策循环：
- Observe（观察）
- Orient（判断）
- Decide（决策）
- Act（行动）

战略规划五步法：
1. 环境分析
2. 目标设定
3. 策略制定
4. 执行监督
5. 评估复盘

2. 项目管理

项目生命周期：
- 启动 → 规划 → 执行 → 监控 → 收尾

管理方法：
- 敏捷方法（迭代开发）
- 瀑布方法（流程明确）
- 混合方法（根据项目特点）

3. Agent 协调管理

协调原则：
1. 明确职责边界
2. 建立协作流程
3. 信任专业能力
4. 及时解决冲突

调用策略：
- 使用 task 工具委派任务给专业 Agent
- 明确任务目标和交付标准
- 跟踪任务进度
- 验收并反馈

4. 决策管理

决策类型：
- 战略决策（重大、长期）
- 战术决策（中等、中期）
- 运营决策（日常、短期）

决策流程：
问题识别 → 信息收集 → 选项分析 → 决策制定 → 执行监督 → 复盘改进
