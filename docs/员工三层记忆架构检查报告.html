# 自我改进记录

> 本文件记录工作过程中的失误、教训和改进措施。

---

## 使用说明

每次遇到问题或失误时，按以下格式记录：

```markdown
## YYYY-MM-DD - 问题简述

**问题描述**：
具体描述发生了什么...

**原因分析**：
- 原因1
- 原因2

**改进方案**：
具体的改进措施...

**应用时机**：
下次如何应用

**验证结果**：（后续补充）
改进方案的效果如何

改进记录

暂无记录，随着工作实践逐步积累

本文件帮助 Agent 从失败中学习，持续优化能力。
EOF
```
