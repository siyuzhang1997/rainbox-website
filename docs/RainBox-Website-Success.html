📈 部署流程

1. 项目创建

mkdir rainbox-website
cd rainbox-website

2. 文件创建

  • ✅ index.html
  • ✅ styles.css
  • ✅ script.js
  • ✅ logo.png
  • ✅ README.md
  • ✅ .gitignore
  • ✅ .github/workflows/deploy.yml

3. Git 初始化

git init
git add .
git commit -m "Initial commit: RainBox official website"

4. GitHub 仓库创建

gh repo create rainbox-website --public --source=. --push

5. GitHub Pages 配置

gh api repos/USER/REPO/pages -X PUT -f build_type=workflow

6. 触发部署

gh workflow run deploy.yml

7. 验证上线

curl -I https://siyuzhang1997.github.io/rainbox-website/
# HTTP/2 200 ✅
