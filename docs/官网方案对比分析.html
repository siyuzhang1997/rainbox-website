B. SSRF 保护机制

Redirect blocked by SSRF protection: 
Access to private/internal IP blocked: 10.88.133.156

原因：
- 目标 URL 解析到私有 IP（10.x.x.x）
- SSRF 保护机制拦截
- 防止服务器端请求伪造攻击

部署方式：Git Push → GitHub Actions → GitHub Pages
